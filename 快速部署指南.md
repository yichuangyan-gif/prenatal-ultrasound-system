## 第三步：获取 GitHub Personal Access Token

如果推送时提示需要密码，按以下步骤获取 Token：

1. 访问：https://github.com/settings/tokens
2. 点击 **Generate new token (classic)**
3. 填写：
   - Note: ` ultrasound-system-deploy`
   - Expiration: 选择有效期（如 90 days）
   - 勾选权限：`repo`（完整仓库访问）
4. 点击 **Generate token**
5. **立即复制显示的 token**（只显示一次！）
6. 推送时用这个 token 作为密码

---

## 第四步：部署到 Render（免费云服务器）

部署到 Render 后，您可以在任何电脑通过浏览器访问系统！

### 操作步骤：

1. 访问：https://render.com/
2. 点击 **Get Started for Free**
3. 选择 **Continue with GitHub**
4. 授权 Render 访问您的 GitHub 账号
5. 点击 **New +** → **Web Service**
6. 找到并选择仓库：`prenatal-ultrasound-system`
7. 配置部署参数：
   ```
   Name: prenatal-ultrasound-system
   Runtime: Python 3
   Branch: main
   
   Build Command: pip install -r requirements.txt
   Start Command: python app.py
   
   Plan: Free
   ```
8. 点击 **Create Web Service**
9. 等待 2-3 分钟部署完成
10. 获得访问链接，例如：
    ```
    https://prenatal-ultrasound-system.onrender.com
    ```

### 访问系统：

部署完成后，您可以在任何电脑的浏览器访问：
```
https://prenatal-ultrasound-system.onrender.com
```

## 第五步：删除 GitHub 仓库中的文件

### 删除敏感文件（venv、数据库等）

双击运行：`delete_from_github.bat`

选择要删除的选项：
- [1] 删除 venv 文件夹（推荐，不应该上传）
- [2] 删除数据库文件
- [3] 删除 __pycache__ 文件夹
- [4] 删除其他文件
- [5] 完全清空仓库（危险！）

### 完全删除整个仓库

1. 访问：https://github.com/yichuangyan-gi/prenatal-ultrasound-system
2. 点击 **Settings**（设置，顶部菜单最右边）
3. 滚动到最底部 **Danger Zone**（危险区域）
4. 点击 **Delete this repository**
5. 输入仓库名称确认：`yichuangyan-gi/prenatal-ultrasound-system`
6. 点击红色按钮确认删除
