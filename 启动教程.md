# 孕期彩超系统启动教程

## 📋 准备工作

### 1. 确认系统已安装
打开文件管理器，检查以下路径是否存在：
```
C:\Users\Administrator\WorkBuddy\prenatal_ultrasound_system
```

如果不存在，说明系统还没安装，需要先安装系统。

### 2. 确认Python已安装
按 `Win + R`，输入 `cmd`，按回车，在命令提示符中输入：
```bash
python --version
```
如果显示版本号（如 Python 3.8.0），说明已安装。如果提示找不到命令，需要先安装Python。

## 🚀 启动系统（详细步骤）

### 方法1：使用启动脚本（推荐）

#### 步骤 1：找到启动脚本
1. 打开文件管理器
2. 进入目录：
   ```
   C:\Users\Administrator\WorkBuddy\prenatal_ultrasound_system
   ```
3. 找到文件：`start.bat`

#### 步骤 2：运行启动脚本
**方法一：双击运行**
1. 在文件管理器中，找到 `start.bat`
2. 双击鼠标左键
3. 会弹出一个黑色命令窗口
4. 等待系统自动安装依赖（第一次需要）

**方法二：右键以管理员身份运行**
1. 在文件管理器中，找到 `start.bat`
2. 右键点击 `start.bat`
3. 选择"以管理员身份运行"
4. 点击"是"允许运行

#### 步骤 3：观察启动过程
启动时，命令窗口会显示：
```
============================================================
孕期彩超检查报告系统
Prenatal Ultrasound Report System
============================================================
系统启动时间: 2024-03-19 14:30:25
访问地址: http://127.0.0.1:8088
============================================================

✅ 系统已更新，支持在Web界面配置API密钥

配置步骤:
1. 打开浏览器访问 http://127.0.0.1:8088
2. 点击右上角的 'API配置' 按钮
3. 输入 OpenAI API 密钥（必需）
4. 输入 Stability AI API 密钥（可选）
5. 点击 '保存配置' 即可使用AI生成功能

注意: API密钥保存在浏览器本地存储，安全可靠

按 Ctrl+C 停止系统
```

**注意**：不要关闭这个黑色窗口！关闭后系统就停止了。

## 🔑 配置API密钥（首次使用必须）

### 步骤 1：找到"API配置"按钮
在页面右上角，你会看到一个按钮："API配置"

### 步骤 2：打开配置窗口
点击"API配置"按钮，会弹出一个窗口。

### 步骤 3：输入OpenAI API密钥
1. 在"OpenAI API密钥"输入框中粘贴你的密钥
2. 密钥格式：`sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`
3. 如何获取密钥：
   - 访问 https://platform.openai.com/api-keys
   - 登录你的OpenAI账号
   - 点击"Create new secret key"
   - 复制生成的密钥

### 步骤 4：输入Stability AI密钥（可选）
如果想使用Stability AI生成更逼真的图像：
1. 在"Stability AI API密钥"输入框中粘贴密钥
2. 如何获取密钥：
   - 访问 https://platform.stability.ai/
   - 注册账号
   - 进入"API Keys"页面
   - 创建新密钥

### 步骤 5：保存配置
点击"保存配置"按钮。

### 步骤 6：验证配置
保存成功后，警告提示会消失。

## 🎨 使用AI生成图像

### 步骤 1：打开AI生成器
在首页点击"AI生成图像"按钮。

### 步骤 2：选择AI模型
在下拉菜单中选择：
- DALL-E 3（推荐，速度快）
- GPT-4o（质量最好）
- Stability AI（医学图像逼真）

### 步骤 3：设置参数
- **孕周**：输入数字（5-42），如 20
- **视图类型**：选择标准视图、胎儿侧面等
- **胎位**：选择头位、臀位、横位

### 步骤 4：生成图像
点击"生成图像"按钮。

### 步骤 5：等待生成
等待10-30秒，AI会生成图像。

### 步骤 6：查看和下载
生成成功后：
- 查看生成的图像
- 点击"下载图像"保存到电脑
- 点击"用于报告"将图像应用到检查报告

## 📌 常见问题解决

### 问题1：无法访问 http://127.0.0.1:8088

**症状**：页面无法打开，显示"无法访问此网站"

**解决**：
1. 检查黑色命令窗口是否还在运行
2. 检查是否显示"系统启动成功"
3. 确认端口号是8088
4. 尝试刷新页面（按F5）
5. 检查防火墙是否阻止了8088端口

### 问题2：AI生成失败

**症状**：点击"生成图像"后提示错误

**解决**：
1. 检查是否已配置API密钥
2. 点击"API配置"检查密钥是否正确
3. 确认密钥以"sk-"开头
4. 检查OpenAI账户余额是否充足
5. 刷新页面重试

### 问题3：密钥保存后无效

**症状**：配置密钥后还是提示未配置

**解决**：
1. 刷新页面（按F5）
2. 清除浏览器缓存
3. 重新输入密钥并保存
4. 检查浏览器是否支持localStorage

### 问题4：页面显示异常

**症状**：页面排版错乱，按钮不显示

**解决**：
1. 刷新页面（按Ctrl + F5强制刷新）
2. 清除浏览器缓存
3. 尝试使用其他浏览器
4. 检查网络连接

### 问题5：命令窗口闪退

**症状**：双击start.bat后窗口立即关闭

**解决**：
1. 右键点击start.bat
2. 选择"以管理员身份运行"
3. 检查Python是否已安装
4. 检查依赖是否已安装：
   ```bash
   pip install -r requirements.txt
   ```
