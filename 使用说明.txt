孕期彩超检查报告系统 - 使用说明
================================

系统位置: C:\Users\Administrator\WorkBuddy\prenatal_ultrasound_system

一、快速启动
============

方法1: 一键启动（推荐）
------------------------
1. 双击运行 start.bat
2. 等待系统自动安装依赖
3. 打开浏览器访问 http://127.0.0.1:8088
4. 开始使用

方法2: 手动启动
----------------
1. 打开命令提示符
2. cd C:\Users\Administrator\WorkBuddy\prenatal_ultrasound_system
3. python app.py
4. 打开浏览器访问 http://127.0.0.1:8088

二、首次使用配置
=================

步骤1: 在Web界面配置API密钥
---------------------------
1. 启动系统后，打开浏览器访问 http://127.0.0.1:8088

2. 点击页面右上角的 "API配置" 按钮

3. 在弹出的配置窗口中输入API密钥：
   - OpenAI API密钥: 从 https://platform.openai.com/api-keys 获取
     用于 DALL-E 3 和 GPT-4o 图像生成
   - Stability AI API密钥（可选）: 从 https://platform.stability.ai/ 获取
     用于生成更逼真的医学图像

4. 点击 "保存配置" 按钮

5. 密钥保存在浏览器本地，立即生效，无需重启

步骤2: 验证配置
---------------
1. 点击首页的 "AI生成图像" 按钮
2. 选择AI模型，设置参数
3. 点击 "生成图像"
4. 如果成功生成图像，说明API密钥配置正确

三、功能模块
=============

1. 患者管理
-----------
- 添加患者: 录入孕妇基本信息
- 查看患者: 查看完整病历
- 编辑信息: 修改患者档案
- 删除患者: 删除患者记录（慎用）

2. 检查报告
-----------
- 创建报告: 为孕妇添加超声检查报告
- 查看报告: 查看报告详情
- 编辑报告: 修改报告内容
- 导出报告: 导出为文本文件

3. AI图像生成
--------------
系统支持三种先进的AI图像生成模型：

a) DALL-E 3 (推荐)
   - OpenAI的成熟图像生成模型
   - 生成速度快，质量稳定
   - 适合日常使用

b) GPT-4o (最新，质量最好)
   - OpenAI最新的多模态模型
   - 图像质量最高，细节更丰富
   - 适合需要高质量图像的场景

c) Stability AI (医学图像更逼真)
   - 专门优化医学图像生成
   - 生成的超声图像更逼真
   - 适合专业医学应用

功能特点：
- 生成图像: 选择AI模型，设置参数，一键生成
- 参数设置: 可设置孕周(5-42周)、视图类型、胎位
- 下载图像: 生成后可下载到本地
- 应用于报告: 生成的图像可直接用于检查报告
- 灰度转换: 自动生成符合超声图像风格的灰度图

5. 数据统计
-----------
- 总患者数: 显示总患者数量
- 检查报告: 显示总报告数量
- 本周检查: 显示本周检查数量
- 平均孕周: 显示平均孕周

四、操作示例
=============

示例1: 添加患者
---------------
1. 点击 "添加患者"
2. 填写基本信息（姓名、病历号为必填）
3. 使用孕期计算器自动计算预产期
4. 点击 "保存患者信息"

示例2: 创建报告
---------------
1. 点击 "新建报告"
2. 选择患者
3. 填写检查信息（检查日期、类型、胎周等）
4. 填写测量数据（BPD、HC、AC、FL等）
5. 填写检查结果和印象
6. 点击 "保存报告"

示例3: AI生成图像
-----------------
1. 点击 "AI生成图像"
2. 设置参数：
   - 孕周: 20
   - 视图类型: 标准视图
   - 胎位: 头位
3. 点击 "生成图像"
4. 等待AI生成图像
5. 下载或用于报告

示例4: 导出报告
---------------
1. 打开报告详情
2. 点击 "导出报告"
3. 选择保存位置
4. 完成导出

五、常见问题
=============

问题1: 系统无法启动
--------------------
原因: Python未安装或依赖未安装
解决:
1. 安装Python 3.8+
2. 运行 pip install -r requirements.txt
3. 重新启动

问题2: AI生成功能不可用
------------------------
原因: API密钥未配置或无效
解决:
1. 点击页面右上角的 "API配置" 按钮
2. 检查 OpenAI API密钥是否正确输入（用于DALL-E 3和GPT-4o）
3. 检查 Stability AI API密钥是否正确输入（用于Stability AI）
4. 确认密钥有效且有足够余额
5. 点击 "保存配置" 并刷新页面

问题3: Stability AI模型无法使用
--------------------------------
原因: Stability API密钥未配置或余额不足
解决:
1. 点击页面右上角的 "API配置" 按钮
2. 在 Stability AI API密钥 输入框中填入密钥
3. 访问 https://platform.stability.ai/ 检查账户余额
4. 新用户通常有免费额度，用完需要充值
5. 点击 "保存配置" 并刷新页面

问题4: 端口被占用
-----------------
原因: 端口8088被其他程序占用
解决:
1. 关闭占用端口的程序
2. 或修改 app.py 中的端口配置
