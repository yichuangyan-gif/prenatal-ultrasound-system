孕期彩超检查报告系统 - 更新日志
================================

版本: 2.0.0
更新时间: 2024-03-19

重大更新
========

1. 移除 .env 文件依赖（彻底更新）
   - 不再需要手动创建和编辑 .env 文件
   - 不再使用 python-dotenv 加载环境变量
   - 后端代码完全移除 os.getenv() 调用
   - 所有API密钥通过HTTP请求头传递
   - 提高系统安全性和易用性

2. Web界面API密钥配置（前端主导）
   - 在页面右上角添加"API配置"按钮
   - 支持可视化输入OpenAI和Stability AI密钥
   - 密钥保存在浏览器本地存储(localStorage)
   - 不会上传到服务器，保护隐私
   - 配置后立即生效，无需重启

3. 后端架构优化
   - 修改 check_api_key 装饰器，从请求头获取密钥
   - 所有AI生成函数改为从 request.headers 获取API密钥
   - 移除 /api/save-api-keys 对 .env 文件的操作
   - 简化 /api/check-api-keys 路由逻辑
   - 删除重复的 Stability AI 路由
   - 更新启动提示信息

4. 文档更新
   - 更新 使用说明.txt，移除 .env 相关说明
   - 添加Web界面配置API密钥的详细步骤
   - 更新常见问题解答
   - 添加版本更新日志
   - 更新系统版本号至 2.0.0

5. 用户体验改进
   - 简化首次使用配置流程
   - 降低非技术用户的使用门槛
   - 提高系统的可移植性和部署便利性
   - 避免.env文件导致的配置错误

配置方式变更（重要）
===================

旧版（1.x）:
   - 需要手动创建 .env 文件
   - 编辑文件填入API密钥
   - 重启系统生效
   - 技术门槛较高

新版（2.0）:
   - 在Web界面点击"API配置"
   - 输入密钥并保存
   - 立即生效，无需重启
   - 简单直观，适合所有用户

技术实现细节
============

前端:
   - localStorage 存储 openai_api_key 和 stability_api_key
   - 发送API请求时在Header中传递密钥:
     * X-OpenAI-Key: OpenAI API密钥
     * X-Stability-Key: Stability AI密钥

后端:
   - 移除: from dotenv import load_dotenv
   - 移除: load_dotenv()
   - 移除: os.getenv('OPENAI_API_KEY')
   - 移除: os.getenv('STABILITY_API_KEY')
   - 改为: request.headers.get('X-OpenAI-Key')
   - 改为: request.headers.get('X-Stability-Key')

兼容性说明
==========

- 完全向后兼容：不影响已有数据和功能
- 纯配置方式变更，不影响核心功能
- 已配置的用户需要重新在Web界面配置一次
- 新用户无需了解 .env 文件概念

迁移指南
========

从1.x升级到2.0:
1. 删除旧的 .env 文件（如有）
2. 启动系统
3. 在Web界面重新配置API密钥
4. 验证AI生成功能是否正常
5. 完成迁移

配置要求
========

必需API:
- OPENAI_API_KEY (用于DALL-E 3和GPT-4o)
- STABILITY_API_KEY (用于Stability AI)

可选:
- 本地Stable Diffusion (需要额外配置)

快速开始
========

1. 配置API密钥
   copy .env.example .env
   notepad .env
   # 填入 OPENAI_API_KEY 和 STABILITY_API_KEY

2. 启动系统
   start.bat
   # 或 python app.py

3. 访问系统
   http://127.0.0.1:8088

4. 使用AI生成
   - 点击"AI生成图像"
   - 选择AI模型
   - 设置参数(孕周、视图、胎位)
   - 点击"生成图像"

注意事项
========

1. API费用
   - OpenAI API按使用量收费
   - Stability AI有免费额度
   - 请注意监控使用量

2. 图像质量
   - 不同模型生成风格有所不同
   - 建议尝试不同模型选择最适合的
   - GPT-4o通常生成质量最高

3. 系统维护
   - 定期备份数据库
   - 更新API密钥时注意保密
   - 建议定期查看API余额

4. 医学应用
   - AI生成图像仅用于教学和研究
   - 不能替代真实医学检查
   - 请遵守相关医疗法规
