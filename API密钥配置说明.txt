API密钥配置功能说明
=====================

更新日期: 2024-01-XX
版本: 1.2.0

新功能介绍
===========

系统现在支持在Web界面上直接配置API密钥，无需手动编辑.env文件！

功能特点
========

1. 可视化配置
   - 在页面右上角点击"API配置"按钮
   - 在弹出的模态框中输入密钥
   - 支持显示/隐藏密钥（点击"显示密钥"复选框）
   - 一键保存，立即生效

2. 密钥本地存储
   - 密钥保存在浏览器本地存储(localStorage)
   - 不会上传到任何服务器
   - 安全可靠，隐私保护
   - 清除浏览器数据会删除密钥

3. 智能检测
   - 自动检测API密钥配置状态
   - 未配置时显示警告提示
   - 配置成功后隐藏警告
   - 支持实时验证密钥格式

4. 即时生效
   - 保存后立即生效，无需重启系统
   - 支持随时修改和更新密钥
   - 支持清除已保存的密钥

配置步骤
========

步骤1: 打开配置界面
--------------------
1. 在页面右上角找到"API配置"按钮
2. 点击按钮打开API密钥配置模态框
3. 界面包含两个输入框：
   - OpenAI API密钥（必需）
   - Stability AI API密钥（可选）

步骤2: 输入OpenAI API密钥
---------------------------
1. 在"OpenAI API密钥"输入框中粘贴密钥
2. 密钥格式: sk-xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
3. 点击"显示密钥"可以查看输入的内容
4. 如果还没有密钥：
   - 访问 https://platform.openai.com/api-keys
   - 登录或注册账号
   - 点击"Create new secret key"
   - 复制生成的密钥

步骤3: 输入Stability AI API密钥（可选）
----------------------------------------
1. 如果希望使用Stability AI生成更逼真的医学图像
2. 在"Stability AI API密钥"输入框中粘贴密钥
3. 点击"显示密钥"可以查看输入的内容
4. 如果还没有密钥：
   - 访问 https://platform.stability.ai/
   - 登录或注册账号
   - 进入"API Keys"页面
   - 创建新的API密钥
   - 新用户有免费额度

步骤4: 保存配置
---------------
1. 确认密钥输入正确
2. 点击"保存配置"按钮
3. 等待系统保存（通常1-2秒）
4. 成功后会显示提示信息
5. 模态框自动关闭
6. 警告提示消失（如果之前显示）

步骤5: 验证配置
---------------
1. 点击首页的"AI生成图像"按钮
2. 选择AI模型（DALL-E 3、GPT-4o或Stability AI）
3. 设置参数（孕周、视图类型、胎位）
4. 点击"生成图像"
5. 如果配置正确，系统会开始生成图像
6. 如果提示密钥错误，请重新配置

注意事项
========

1. 密钥格式验证
   - OpenAI密钥必须以"sk-"开头
   - 系统会自动验证密钥格式
   - 格式不正确时会提示错误

2. 密钥安全
   - 密钥仅保存在浏览器本地
   - 建议不要在公共电脑上使用此功能
   - 使用后建议清除浏览器数据
   - 定期更换密钥提高安全性

3. 浏览器兼容性
   - 支持所有现代浏览器
   - 需要启用JavaScript
   - 支持Chrome、Firefox、Safari、Edge

4. 故障排除
   - 如果密钥保存后无法使用：
     a. 检查密钥是否有效
     b. 检查账户余额是否充足
     c. 尝试重新输入并保存
     d. 检查浏览器控制台是否有错误

高级功能
========

1. 清除密钥
   - 在API配置模态框中删除密钥内容
   - 点击"保存配置"
   - 密钥将从本地存储中删除

2. 更新密钥
   - 随时打开API配置界面
   - 修改密钥内容
   - 点击"保存配置"
   - 新密钥立即生效

3. 备份密钥
   - 点击"显示密钥"查看完整密钥
   - 复制并保存到安全的地方
   - 建议定期备份

配置示例
========

场景1: 首次使用系统
--------------------
1. 打开系统，看到警告提示"API密钥未配置"
2. 点击"API配置"按钮
3. 输入OpenAI API密钥
4. （可选）输入Stability AI API密钥
5. 点击"保存配置"
6. 开始使用AI生成功能

场景2: 密钥失效或过期
--------------------
1. AI生成功能提示密钥错误
2. 点击"API配置"按钮
3. 删除旧密钥，输入新密钥
4. 点击"保存配置"
5. 重新生成图像

场景3: 切换不同的AI模型
------------------------
1. 确保已配置OpenAI密钥（必需）
2. 如果使用Stability AI，确保已配置其密钥
3. 在AI生成界面选择模型
4. 点击"生成图像"

常见问题
========

Q1: 密钥保存在哪里？
A: 密钥保存在浏览器本地存储(localStorage)，不会上传到服务器。

Q2: 清除浏览器数据会删除密钥吗？
A: 是的，清除浏览器缓存和Cookie会删除本地存储的密钥。

Q3: 可以在不同电脑上使用吗？
A: 密钥保存在本地，换了电脑需要重新配置。

Q4: 如何备份密钥？
A: 在API配置界面点击"显示密钥"，然后复制保存到安全的地方。

Q5: 密钥安全吗？
A: 相对安全，因为只保存在本地。但建议在可信的设备上使用。

Q6: 为什么保存后还是提示未配置？
A: 可能的原因：
   - 密钥格式不正确
   - 浏览器禁用了本地存储
   - 需要刷新页面

Q7: 可以同时配置多个密钥吗？
A: 可以，OpenAI密钥（必需）和Stability AI密钥（可选）可以都配置。

Q8: 如何知道密钥是否有效？
A: 尝试生成图像，如果成功则说明密钥有效。
